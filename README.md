# Automotive Market Intelligence Scraper

Production-oriented automotive market data extraction pipeline built with Python and Selenium-Stealth.

## Features

- Enterprise Selenium scraping
- Selenium-Stealth anti-bot
- Multi-threaded scraping
- Automated scrolling
- Structured vehicle extraction
- Google Sheets integration
- Excel export
- Logging system
- Data cleaning pipeline
- SKU generation system
